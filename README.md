## üìå Features

### For Users
- Browse meals from local chefs.
- View detailed meal information including ingredients, chef, availability, and delivery time.
- Place orders through a modal form.
- Add meals to favorites.
- Leave reviews on meals.
- Responsive and modern UI with Framer Motion animations.

### For Chefs
- View and manage your meals.
- Receive and track orders in real-time.

### General
- Firebase Authentication for secure login and user management.
- MongoDB for storing users, meals, orders, reviews, and favorites.
- Stripe integration for payment processing.
- Fully responsive design for mobile and desktop.
- Premium UI with animations and smooth user experience.
